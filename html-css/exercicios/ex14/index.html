ID e CLASSES

Agora bora aprender um bagulho sinistro, que é atribuição de ID e CLASSES.
Me parece um papo mais interessante... Pelo menos mais interessante que teoria das cores e fontes.

ID

O id é um indentificador de estilos especiais, que você define no CSS externo ou interno. Você cria esse estilo "especial", define um nome pra ele usando a "#", tipo assim: #estilo-especial, e esse nome se torna o identificador. Depois pra chamar no HTML é só usar: id="estilo-especial".

Esse aqui é um h3 com CSS definida padrão

O código com CSS padrão de h3:


h3{
  margin-left: 20px;
  color: #1800DC;
}

E esse aqui é um h3 com CSS especial e identificador :)

O código do CSS com o identificador:

  
#especial{
  font-size: 50px;
  font-family: sans-serif;
  margin-left: 30px;
  color: #000564;
}

Código do h3 com o id:


        <h3 id="especial">E esse aqui é um h3 com CSS especial e identificador :)</h3>

Rapais, fica esperto que você só pode usar um id por documento HTML... Então se já usei o id="especial" no h3 ali em cima, não posso usar esse mesmo id="especial" em outro h3, h1, h2, ou qualquer outra tag dos inferno que seja. Na real até pode, faça o que você quiser, siga seus sonhos, reaja a assaltos, mas é ERRADO. Nada me impede de criar outra id e botar em outro canto... Mas não vai socar id no HTML inteiro também né, meio sem noção.

Classes

Pra suprir essa "desvantagem" do id, existe as famosas classes.
Uma classe é basicamente o mesmo conceito do id, tu vai definir na CSS qual estilo essa classe vai ter, depois tu soca classe onde você quer usar no HTML.
Uma pequena diferença entre os dois é que o id você define o nome com "#" e a classe você define com um ".", e na hora de chamar a classe tu bota class="exemplo"

UM H4 SEM CLASSE

Código de CSS do h4 sem o class:


h4{
  margin-left: 15px;
  color: #8378F5;
}

UM H4 COM CLASS

Código do class na CSS:


.classudo{
  border: 1.5px solid #78E4F5;
  color: #5000F5;
  font-size: 30px;
  font-family: "Roboto", sans-serif;
}
.classudo::after{
    content: "\01F9D0";
}

Código do h4:


      <h4 class="classudo">UM H4 COM CLASS</h4>

Da pra botar duas classes ao mesmo tempo num mesmo bagulho e até mesmo misturar id com classes, mas como já usei o id lá em cima vou mostrar só duas classes juntas porque eu sou um bom moço e não faço coisas erradas...
TIPO ESSE AQUI, TEM DUAS CLASSES.

O código fica assim: 

<span class="classudo exemplo">TIPO ESSE AQUI, TEM DUAS CLASSES.</span>
Código da classe exemplo é esse: 

.exemplo{
  font-weight: bold;
  text-decoration: underline;
}

Note que eu usei um emojizim ali pra dar uma diferenciada, pra isso tive que fazer a classe ter outra classe recebendo um elemento no final.

Pseudo-classes

Agora que você sabe o que é ID e o que é class, vamos falar de pseudo-classes e pseudo-elementos.
Uma pseudo-classe tem que tá ligada a uma classe ou a um elemento da CSS, porque ela vai referenciar e mudar o que ela tiver que mudar daquela classe ou elemento. Normalmente o que a pseudo-classe faz é mudar o estado e estilizar uma CSS já feita, tipo quando passa o mouse por cima e acontece alguma coisa, clica num link e ele muda de cor depois que mudou, esse tipo de coisa. Pra nomear uma pseudo-classe você usa ":". Vamo mostrar uns exemplos como sempre.

Pseudo-classe:hover
Aqui temos uma tag <div> que vai receber os estilos de uma pseudo-classe chamada hover, que só aplica o estilo quando você passa o mouse em cima... ENTÃO ESCURREGUE O MOUSE AQUI.

O código da div tá assim: 

div{
  width: 300px;
}
O código da CSS fica assim da div:hover: 

div:hover{
  color: magenta;
  background-color: grey;
  width 200px;
  margin-top: 10px;
}
Pseudo-classe:visited e pseudo-classe:active

Aqui vou te mostrar interação de pseudo-classes com links.
Com a pseudo-classe visted o link abaixo vai aparecer de cor diferente quando você já tiver visitado o link antes, e com o estado active, vai mudar de cor depois que você clicar. Faça seus testes ai.

Clique aqui pra ver meu estado de espirito nesse momento
Como você deve ter imaginado, pra usar essas duas pseudo-classes especificas pra links, no css tu vai atribuir elas na tag <a>.
Código do link: 

<a href="https://www.youtube.com/watch?v=16xGceArnz8" target="_blank">Clique aqui pra ver meu estado de espirito nesse momento</a>
Código da CSS de <a> 

a:hover{
  color: red;
}
a:visited{
  color: magenta;
}
a:active{
  color: darkred;
}

Tem também os pseudo-elementos, que inclusive você já viu aqui nesse documento, que é o tal do .classudo::after que eu usei pra botar um elemento no final de tudo que usar a classe classudo. Enquanto as pseudo-classes mexem no estado e a interação de um conteúdo, o pseudo elemento mexe no conteúdo em si, podendo acrescentar e modificar coisas da classe onde ele estiver referenciando.
Vamo pra outros exemplos de pseudo-elemento...

::before & ::after

AQUI É BODY BUILDER POR#@ BIIIRL

Código do <span>: 

<span class="rubik-storm-regular">AQUI É BODY BUILDER POR#@ BIIIRL</span>
Código da CSS (tive que importar fonte e tem link no head também):


<head>
  <link rel="stylesheet" href="style.css">
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
  <link href="https://fonts.googleapis.com/css2?family=Rubik+Storm&display=swap" rel="stylesheet">
</head>

.rubik-storm-regular {
  font-family: "Rubik Storm", system-ui;
  font-weight: 400;
  font-style: normal;
}
.rubik-storm-regular::before{
  content: "\01F4AA";
}
.rubik-storm-regular::after{
  content: "\01F3CB";
}
::first-line
