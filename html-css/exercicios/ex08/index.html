    Exemplos

    HTML
    O que cria o conteudo e marcacoes de um site (a estrutura de uma musica, a partitura, a strofe, o refrao, o climax).
    CSS
    O que estiliza o site, faz o design (e a performance e interpretacao da musica, o que faz ela ser bonita, tocante e inspiradora, sao as dinamicas, as variacoes de ritmo, a estetica, as nuances).
    JavaScript
    E a linguagem de programacao que faz a iteratividade do site, que manda em como ele vai funcionar (e a teoria musical pura, as as regras de contraponto, regras de harmonia, regras de escrita).
