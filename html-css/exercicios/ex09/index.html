Usando links externos

Da pra transformar qualquer palavra em um link, por exemplo: Batata ← e um link pra explicacao de batata na Wikipedia.
Pra fazer isso, tem que fazer uma ancora na palavra com a tag "<a>": <a href="URL">Batata</a>.
Isso cria um link que abre a pagina na mesma aba do site atual (nao recomendado).

Link para outra aba

Pra criar um link que abra uma nova aba no navegador, tem que usar a referencia target="_blank", e se quiser usar uma referencia semantica e SEO, da pra usar: rel="external"
Exemplo: Vou criar um link pra receita de sopa de batata. (<a href="URL" target="_blank" rel="external">sopa de batata</a>).

Usando Links internos

Os links internos servem mais pra direcionar para coisas internas do site, pra navegar dentro de arquivos do proprio servidor (da minha bagunca interna no caso).

Clique aqui para ver a magica!
No caso, basta fazer o href receber o caminho do arquivo pra onde voce quer que o link leve.
Exemplo do link la ↑:
Clique <a href="pag002.html">aqui</a> para ver a magica!

Link para download

Aqui supostamente tem que aprender a criar links para baixar coisas, basta usar o caminho do arquivo que quer baixar com href, e usar o parametro download para baixar, tudo dentro da tag ancora.
E tiroteio, ABAIXAR

Nesse caso como era so uma imagem, ele apenas abre no navegador, mas se quiser baixar um arquivo, faz a mesma coisa so que com a pasta:
ABAIXA
O codigo fica assim: <a href="abaixa.zip" download="abaixa.zip">ABAIXA</a>
